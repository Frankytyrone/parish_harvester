# ============================================================
# Derry Diocese — Known Bulletin URL Evidence Log
# ============================================================
# Purpose: This file records REAL, manually verified bulletin URLs
# found on each parish website.  The harvester code reads this file
# to make mathematical predictions about where THIS WEEK's bulletin
# will be located — by detecting the date pattern and shifting it.
#
# HOW IT WORKS (The "Safe Cracker"):
#   Each parish uses one of these URL date patterns:
#     Pattern A  →  DDMMYY       e.g. carndonaghparish.com/pdf/120426.pdf
#     Pattern B  →  D-M-YY       e.g. limavadyparish.org/onewebmedia/5-4-26.pdf
#     Pattern C  →  YYYY-MM-DD   e.g. clonmanyparish.ie/wp-content/uploads/2026/04/2026-04-12.pdf
#     Pattern D  →  DD-Month-YYYY slug  e.g. bellaghyparish.com/.../Newsletter-12-April-2026-1.pdf
#     Pattern E  →  [YYYY-M-D]   e.g. greenlough.com/.../[2026-4-12].pdf
#     Pattern F  →  Static URL   e.g. laveyparish.com/.../laveyparishbulletin.pdf (never changes)
#     Pattern G  →  HTML page / WP post slug  e.g. clonleighparish.com/2026/04/03/strabane-...
#     Pattern H  →  Banagher numeric ID  e.g. banagherparish.com/files/9/Newsletters/384/Bulletin-...
#
# FORMAT:
#   One URL per line.  Blank lines and lines starting with # are ignored.
#   Group entries by parish name.  Most recent bulletin first.
#
# SPECIAL NOTES:
#   - desertmartinparish.com  →  STALE (last bulletin March 2025), do not use
#   - urneyandcastlefinparish.com  →  DEAD URL, do not use
#   - staroftheseacatholic.com  →  SSL error on main site; bulletin is on Google Drive (see below)
#   - iskaheenparish.com  →  bulletin is a JPEG image, not a PDF
#   - cappaghparish.com/pdf/120426.pdf  →  URL looks current but content is from 2025, ignore
# ============================================================
