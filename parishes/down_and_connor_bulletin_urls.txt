# ============================================================
# Down and Connor Diocese — Known Bulletin URL Evidence Log
# ============================================================
# Purpose: This file records REAL, manually verified bulletin URLs
# found on each parish website.  The harvester code reads this file
# to make mathematical predictions about where THIS WEEK's bulletin
# will be located — by detecting the date pattern and shifting it.
#
# HOW IT WORKS (The "Safe Cracker"):
#   Each parish uses one of these URL date patterns:
#     Pattern A  →  DDMMYY       e.g. holycrossparishbelfast.com/pdf/120426.pdf
#     Pattern D  →  DD-Month-YYYY / slug variants
#                   e.g. ballycastleparish.com/s/12th-April-2026.pdf
#     Pattern F  →  Static URL   e.g. carrickparish.org/_files/ugd/... (never changes)
#     html_link  →  HTML page, no predictable PDF URL — link shown in mega PDF
#     image      →  Bulletin is a JPEG/PNG image, converted to PDF
#
# FORMAT:
#   One URL per line.  Blank lines and lines starting with # are ignored.
#   Group entries by parish name.  Most recent bulletin first.
# ============================================================
